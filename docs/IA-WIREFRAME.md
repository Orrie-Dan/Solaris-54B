# Solaris 54 IA Wireframe (Corporate v1)

## Route Map
- `/`: Executive overview and primary conversion
- `/solutions`: Seven-service overview and deep links
- `/solutions/:slug`: Service detail page template
- `/ssit`: Smart Solar Intelligent Technology platform page
- `/markets`: Global regions and expansion focus
- `/group`: Group structure and shared market strategy
- `/contact`: Enquiry and qualification page

## Global Component Inventory
- Top nav with bilingual toggle and primary CTA
- Footer with group attribution
- Proof strip (group backing, standards, global reach, response SLA)
- Reusable card primitives (service, region, differentiator)
- CTA band
- Stat chips
- Route-level SEO metadata

## Home Information Hierarchy
1. Hero value proposition (developer-led lifecycle).
2. Proof strip (credibility markers).
3. Audience tiles (government, utility, enterprise, digital/telecom).
4. Solutions preview with SSIT labels.
5. Why Solaris 54 differentiators.
6. CTA band to contact.

## Typography Rules (Corporate)
- Headings: Syne 700/800 for restrained display use.
- Body: Inter 400/500, avoid light weight for readability.
- Labels/meta: JetBrains Mono only for badges/tags.
- Section rhythm: 80px desktop, 56px mobile vertical spacing.
